# Contributing to TimeClaw

TimeClaw is designed for **OpenClaw** and optimized for **agent-first** usage.

## Development

Requirements:
- Node >= 18

Run tests:

```bash
npm test
```

## What to work on

- Check **GitHub Projects** for the roadmap.
- Look for issues labeled **good first issue** or **help wanted**.

## Conventions

- Keep changes small and focused.
- Prefer cross-platform code (Windows/macOS/Linux).
- Update docs (README / SKILL.md) when behavior changes.

## Security

If you believe you found a security issue, please follow `SECURITY.md`.
